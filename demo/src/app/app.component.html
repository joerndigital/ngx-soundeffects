ngx Soundeffects

Directive ngxSound

<button
ngxSound
[soundFile]="'coin.wav'"
[soundVolume]="0.8"
>

For a button click to be accompanied by music, the directive must be included as shown on the left.

<button
ngxSound
[soundFile]="'coin.wav'"
[soundVolume]="0.8"
[soundOnClick]="false"
[soundOnHover]="true"
>

To trigger the sound on a mouse hover, [soundOnHover]="true" can be used. There are more inputs, e.g. to pause or start the sound, adjust the volume or others.

Decorator @Sound()

@Sound('coin.wav')
public soundCheck()

You can see the simplest application of the Decorator on the left. As a parameter you simply enter the file name with extension.

@Sound({{ "{" }}
  fileName: 'coin',
  extension: '.wav'
{{ "}" }})
public soundCheck()

An object can also be passed as a parameter. The properties can be found further down in the section "ISoundDecorator". More properties should be added over time.

@Sound({{ "{" }}
  start: ISoundDecorator,
  success: ISoundDecorator
  error: ISoundDecorator
{{ "}" }})
public soundCheck()

The whole thing also works asynchronously. Here you can pass a start, success and error sound.

@Sound({{ "{" }}
  start: ISoundDecorator,
  success: ISoundDecorator
  error: ISoundDecorator
{{ "}" }})
public soundCheck()

And here is another example with error sound.
